# PlantGuard AI 🌱

PlantGuard AI is an AI-powered plant disease detection and farming assistant designed to help farmers and plant enthusiasts identify plant diseases and receive guidance for treatment and prevention.

## Features

* Upload plant leaf images for disease detection
* AI-powered disease prediction using CNN
* Disease explanation and causes
* Treatment recommendations
* Prevention tips
* Plant care chatbot assistant

## Tech Stack

### Frontend

* React
* Vite
* TypeScript
* Tailwind CSS

### Backend

* Flask
* Python

### Machine Learning

* CNN (Convolutional Neural Network)
* TensorFlow / Keras

## Installation

### Clone Repository

```bash
git clone https://github.com/Priyan-V112/plantguard-ai.git
cd plantguard-ai
```

### Install Frontend Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm run dev
```

### Run Backend

```bash
python bot.py
```
